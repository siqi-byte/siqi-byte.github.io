Mac Folder Multilingual Settings

Posted on 2019-11-25 In mac 631 1 mins.

The chinese dir of finder always needs to switch between Chinese and English input continuously under terminal.
This method can create an English dir and display it in Chinese.

close Rootless

  • shutdown && cmd + R
  • csrutil disable
  • restart computer

backups && modify

  • cd /System/Library/CoreServices/SystemFolderLocalizations/zh_CN.lproj
  • sudo cp SystemFolderLocalizations.strings SystemFolderLocalizations.strings.bak
  • sudo plutil -convert json SystemFolderLocalizations.strings
  • add key => value information
  • sudo plutil -convert binary1 SystemFolderLocalizations.strings
  • cd ~/targetFolder
  • touch .localized
  • killall Finder

It’s over!
